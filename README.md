# ボンバーマン（仮）
あの、懐かしのボンバーマンが、、、C#コンソールで帰ってきた！？！？  
待望の熱いバトルが今、始まる...

# TL;DR
* ボンバーマンを操作して💣を配置し、`↑W` `←A` `↓S` `→D`で操作して敵を倒すっていうゲームです
* もちろん対戦はないです。
* 敵は不動

# 遊び方
1. visualstudioでプロジェクトを開く
1. `↑W` `←A` `↓S` `→D`で移動
1. 爆弾💣を置くにはPress any key

# 前提条件
* VisualStudio installed　　
* .NET Runtime
